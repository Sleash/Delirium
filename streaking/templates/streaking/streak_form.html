What's EBSI ?

Eden's Blessing Stack Index, or EBSI, is a value defined as the maximum amount of Eden's Blessings collected in a single run of a given streak, between 0 and 10. Typical values can be :

  • 10 for gamebreaking players.
  • 1 or 2 for regular non-gamebreaking gameplay.
  • 0 for extra challenge streaks where each run is self-contained.
Providing an EBSI value is optional and any value is of course allowed. It merely serves as an extra at will restriction, as Eden's Blessing's unique mechanic can be considered overpowered in a game of independant runs.
